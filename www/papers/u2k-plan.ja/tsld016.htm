Multiple Coded Scheme

  • editable entity unit (eeu)

    • 文字を抽象化したもの
      • 文字符号 = eeu の id
    • 各種文字符号や各種属性を持つ
      • 文字符号を持たない eeu を可能にする
        • 文字以外の情報
        • 符号化されていない文字
        • 文字符号に合わない文字の用法
      • SGML / XML 等の tag 付き文書
      • 文字データベースの利用や編集
